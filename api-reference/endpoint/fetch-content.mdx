### Fetch memory contents

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X 'POST' \
    'https://api.hydradb.com/fetch/content' \
    -H 'accept: application/json' \
    -H 'Content-Type: application/json' \
    -d '{
    "tenant_id": "string",
    "sub_tenant_id": "string",
    "source_id": "string",
    "mode": "content",
    "expiry_seconds": 3600
    }'
    ```
    
  </Tab>
  <Tab title="TypeScript">
    ```ts
    const content = await client.fetch.content({
      source_id: "HydraDoc1234",
      tenant_id: "tenant_1234",
      sub_tenant_id: "sub_tenant_4567",
      mode: "content",
      expiry_seconds: 3600
    });
    ```
  </Tab>
  <Tab title="Python (Sync)">
    ```python
    # Async usage is similar, just use async_client and await
    content = client.fetch.content(
        source_id="HydraDoc1234",
        tenant_id="tenant_1234",
        sub_tenant_id="sub_tenant_4567",
        mode="content",
        expiry_seconds=3600
    )
    ```
  </Tab>
</Tabs>

### Functionality

- Extracts user credentials and organization details from the API key
- Validates the file access permissions for the specified tenant
- Creates secure access tokens for file retrieval
- `mode="content"`: Returns the actual file content directly
- `mode="url"`: Generates a time-limited download URL for the requested file
- `mode="both"`: Returns both content and a download URL
