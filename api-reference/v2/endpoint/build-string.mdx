## Overview

`buildString` (TypeScript) / `build_string` (Python) formats a `POST /search` result into a plain string you can pass directly to an LLM.

It works with any result from `client.search.query()` regardless of `source` or `search_by`.

<Info>
This helper is part of the SDK only. It makes no network calls.
</Info>

---

## Knowledge only

<CodeGroup>
```typescript TypeScript SDK
import { HydraDBClient, buildString } from "@hydradb/sdk";

const client = new HydraDBClient({ token: process.env.HYDRA_DB_API_KEY });

const result = await client.search.query({
  tenant_id: "my_tenant",
  query: "What is our refund policy?",
  source: "knowledge",
  search_by: "hybrid",
  graph_context: true,
});

console.log(buildString(result));
```
```python Python SDK
from hydra_db import HydraDB, build_string

client = HydraDB(token="YOUR_API_KEY")

result = client.search.query(
    tenant_id="my_tenant",
    query="What is our refund policy?",
    source="knowledge",
    search_by="hybrid",
    graph_context=True,
)

print(build_string(result))
```
```python Python SDK (Async)
from hydra_db import AsyncHydraDB, build_string

client = AsyncHydraDB(token="YOUR_API_KEY")

result = await client.search.query(
    tenant_id="my_tenant",
    query="What is our refund policy?",
    source="knowledge",
    search_by="hybrid",
    graph_context=True,
)

print(build_string(result))
```
</CodeGroup>

## Output format

Each chunk is rendered with its source, content, and graph relations inline:

```text
=== CONTEXT ===

Chunk 1
Source: refund-policy.pdf (score: 1.33)

Refunds are issued within 30 days of purchase. To request a refund,
contact support with your order number and reason for the request.

Graph Relations:
  [refund policy] -> GOVERNS -> [support team]: The support team processes all refund requests.
  [refund policy] -> RELATED_TO -> [billing department]: Approved refunds are processed by billing.
---

Chunk 2
Source: refund-policy.pdf (score: 1.21)

Refunds are not available for digital products once downloaded.
---
```
